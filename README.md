## Week 2 LU分解法求解矩阵

LU分解法是一种常用的线性代数方法，在求解矩阵方程、解线性方程组等问题时广泛应用。它将一个矩阵分解为一个下三角矩阵L和一个上三角矩阵U的乘积，即A=LU，然后通过前代和回代求解方程组，从而得到未知数的解。
